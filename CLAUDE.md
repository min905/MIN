## 프로젝트 개요

트렌드리포트 — 트렌드 데이터를 수집하고 리포트로 제공하는 프로젝트.

## 기술 스택

- **런타임**: Node.js
- **패키지 관리**: npm

## 시작하기

```bash
npm install
npm start
```

## 개발 명령어

```bash
npm run dev      # 개발 서버 실행
npm test         # 테스트 실행
npm run build    # 빌드
```
